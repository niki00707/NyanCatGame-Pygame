# 🌈 Nyan Cat Game

Аркадная 2D-игра, разработанная на Python с использованием библиотеки Pygame.

Игрок управляет перонажем "Nyan Cat", собирает "хорошую" еду, избегает "плохих" объектов и проходит уровни возрастающей сложности.

---

# ✨ Особенности игры

- 🎮 3 уровня сложности
- 🐱 Анимированный персонаж
- 🍔 Система полезной и вредной еды
- ❤️ Система жизней
- ⭐ Бонусные звезды счастья
- 🔊 Музыка и звуковые эффекты
- ⚙️ Меню настроек
- ⏸️ Система паузы

# ⚙️ Установка и запуск

## 1. Клонирование репозитория

```bash
git clone https://github.com/niki00707/NyanCatGame.git
```

---

## 2. Установка библиотеки Pygame

```bash
pip install pygame
```

---

## 3. Запуск игры

```bash
python main.py
```

---

# 🎯 Цель игры

Собирайте "хорошую" еду, избегайте "плохой". Пройдите все уровни сложности, не потеряв все сердца.
