# 🌈 Nyan Cat Game

Аркадная 2D-игра, разработанная на Python с использованием библиотеки Pygame.

Игрок управляет Nyan Cat, собирает полезную еду, избегает опасных объектов и проходит уровни возрастающей сложности.

---

# ✨ Особенности игры

- 🎮 3 уровня сложности
- 🐱 Анимированный персонаж
- 🍔 Система полезной и вредной еды
- ❤️ Система жизней
- ⭐ Бонусные звезды счастья
- 🔊 Музыка и звуковые эффекты
- ⚙️ Меню настроек
- ⏸️ Система паузы
